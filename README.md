# Shoe E-Commerce Website

A simple e-commerce demo website for browsing and “buying” shoes.

## What this project does
- Displays a list of shoes/products  
- Allows adding items to cart  
- Provides a simple checkout flow  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Features
- Product catalog  
- Add to cart  
- Basic checkout simulation
